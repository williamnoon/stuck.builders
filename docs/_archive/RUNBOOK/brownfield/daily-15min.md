# Brownfield Daily — 15 min planning-mode session (Days 2-6)

Source-of-truth: `docs/PRD.md` §3B · memory rule `feedback_delivery_style_they_code.md` (updated 2026-07-10).

## Prerequisites (agents automate these)

- [ ] Sprinter's overnight agent has committed to a branch (never main)
- [ ] Agent has produced an overnight recap with: (a) target status, (b) commits, (c) taste calls flagged, (d) blockers if any
- [ ] Radar has ingested last 24 hrs of sprinter's transcripts
- [ ] Fuse (if member) has fetched relevant community skills for today's plan

## Session structure (15 min)

### 0:00 — 0:02 · Load today's plan
- Both open the daily plan artifact
- Review yesterday's outcome flag (target hit / partial / stuck)

### 0:02 — 0:05 · Show what agent shipped overnight
- Sprinter shows the branch's commits
- Show the overnight recap the agent produced

### 0:05 — 0:10 · Review agent-flagged taste calls
- Read each flagged call
- Sprinter decides (Will guides): approve / redirect / defer
- Answer flows back to agent for the next run

### 0:10 — 0:13 · Any unstick topic sprinter brings
- Sprinter's turn to raise anything
- Will responds; no line-by-line typing

### 0:13 — 0:15 · Confirm tonight's plan + close
- Confirm what agent does tonight
- Agree on tomorrow's 15-min window

## What Will does vs the sprinter

**Will:**
- Talks about outputs and decisions
- Answers taste calls the agent can't make
- Copilots the orchestration: "your agent should try X" not "let me type this"

**Sprinter:**
- Talks about their reads on what the agent shipped
- Makes the calls (Will guides, sprinter decides)
- Directs their agent post-call for tonight's work

**Never:**
- Line-by-line typing on the call
- Will making decisions on the sprinter's behalf
- Agent doing anything on main or prod without approval

## Post-session (auto)

- [ ] Sprinter taps: target hit / partial / stuck → updates daily plan
- [ ] Radar surfaces skill candidates from today; sprinter approves for personal library
- [ ] Tomorrow's plan auto-adjusts based on today's outcome
- [ ] Transcript saved to sprinter's Radar archive
- [ ] If opt-in: anonymized wall pattern contributes to community aggregation

## Unstick calls between sessions

Up to 5 across the week, up to 10 min each. Sprinter triggers via async line; Will responds within a few hours (usually much faster).
